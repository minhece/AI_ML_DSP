RADAR AND ULTRASOUND HYBRID SYSTEM FOR HUMAN COMPUTER INTERACTION (M.S THESIS)

June 2017 - Texas Tech University - Supervised by Prof. Changzhi Li

Designed a hybrid system using FMCW RADAR (TI AWR1443) work at 77 GHz and ultrasound system work at 40kHz. The radar can detect hand movement and control computer. The system can detect waving hand left, right, up, down, single tap, double tap. The ultrasound will detect at near distance (<30 cm) and radar detect at far distance (30 cm < l < 120 cm)
